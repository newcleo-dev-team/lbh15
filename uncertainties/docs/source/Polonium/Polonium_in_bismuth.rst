===================
Polonium in bismuth
===================

The activity coefficient for dilute polonium in bismuth in the temperature interval 723-1123K follows the expression :cite:`book9` :cite:`book15`:

.. math:: 
   \gamma_{Po(Bi)} = 10^{-\frac{2272.7}{T} + 0.1316}
   :label: gamma_po_bi
